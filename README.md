# cpp-learning

### 01-Task
Задача 1: первый опыт с классами.

Реализовать класс целое положительное число. В нем должен храниться массив в C-стиле с делителями этого числа. Причем, число бывает двух видов: с отсортированным массивом и нет, необходимо поддерживать оба вида.

Соблюсти инкапсуляцию, сделать корректный “обвес”: конструкторы (копирования и обычный), деструкторы, желательно присваивание (но это опционально) и прочее. Дополнительно должны быть методы по нахождению НОК и НОД для пары чисел (с использованием массива делителей), реинициализации. 

Наконец, хочется такой операции: изменить делитель X на Y. Если у числа такого делителя нету — ничего делать не надо, иначе — надо его вписать и пересчитать число. Сделать из этого всего модуль.

Обязательно должна быть демка: в main мы работаем этими числами и что-то с ними делаем (показываем что все работает). Makefile обязателен, используем google style guide.

Замечание: мы пользуемся тем, что было рассказано на паре. Можно сделать более продвинуто, но тогда надо объяснять, что происходит.
